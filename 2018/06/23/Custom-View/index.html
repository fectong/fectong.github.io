自定义 View

Fe2Cu
2018-06-23
Android, Custom View, View

WHAT

自定义 View 通常是 View 的子类; 也可以继承自已实现的 View 子类;
自定义 3D View 通常是 SurfaceView 的子类

WHEN

当已有控件不满足当前需求时, 即可尝试使用自定义 View

HOW

通常有以下几个步骤:

  • 创建 View 的子类
  • 添加自定义 View 属性
  • 应用自定义属性
  • 添加属性方法
